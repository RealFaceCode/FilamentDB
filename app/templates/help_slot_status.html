{% if lang == 'de' %}Empfehlung für Live-Daten{% else %}Recommended live-data setup{% endif %}

    {% if lang == 'de' %}
  1. Bild 1 öffnen: Auf /slot-status den Status aller Slots prüfen.
  2. Live-Daten sichern: Lokalen Bridge-Dienst im Drucker-LAN starten (`local_services/local_slot_bridge.py`) und Push zu `POST /api/slot-state/push` aktiv halten.
  3. Abweichungen beheben: Mismatch/Missing/Stale nacheinander korrigieren, bis alle kritischen Slots wieder OK sind.
    4. {% else %}
  4. Open Image 1: Review all slot states on /slot-status.
  5. Keep live data flowing: Run the LAN bridge (`local_services/local_slot_bridge.py`) and keep pushes to `POST /api/slot-state/push` active.
  6. Resolve deviations: Fix mismatch/missing/stale items until critical slots return to OK.
    7. {% endif %}
